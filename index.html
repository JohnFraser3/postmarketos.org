postmarketOS





It is 2017. Pick an average PC from 2007 and install a minimal Linux based operating system. You will be able to do basic computing tasks (eg. surfing the web, reading E-Mails, listening to music, chatting) just like on an expensive modern PC. You will even get security updates, so your old computer is protected, just like as a new one. postmarketOS aims the same on smartphones also, Also to help reduce atleast a fraction of the e-waste accumulated on the planet.
